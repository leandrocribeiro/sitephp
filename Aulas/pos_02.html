1. Layout

Quase tudo é widget: Text, Image, Icon, Center, Column, Row, Container.

A app é um widget: 

void main() {
  runApp(const MainApp());
}

O método build() retorna um Widget que pode conter outros widgets.

1.1. Scaffold

O widget Scaffold fornece banner, cor de fundo, a outros componentes úteis à uma MaterialApp.

1.2. Container

Container pode receber padding, decoration, margin e possui um child.

Sem nenhuma propriedade ajustada ele é inútil.

Cada propriedade tem seus tipos de valores e nem sempre corresponde ao que usamos em CSS.

Child deve ser a última propriedade. Quick Fix! 

body: Container(
	padding: const EdgeInsets.all(50),
	child: const Text('Soma'),
),

decoration é ajustada com BoxDecoration que fornece uma caixa. 

body: Container(
	decoration: BoxDecoration(
	  shape: BoxShape.circle,
	  color: Colors.yellow,
	  border: Border.all(color: Colors.blue, width: 5),
	),
	padding: const EdgeInsets.all(50),
	margin: EdgeInsets.only(left: 20),
	child: const Text('Soma'),
),

1.3. Center

Center centraliza o filho (child).

1.4. Column

Vamos criar uma tela com três atividades:

  • Estudar para prova de matemática.
  • Campeonato de futebol.
  • Festa da Joana.

A barra da app ficará assim: 

appBar: AppBar(
	title: const Text('Atividades'),
),

O Widget Center não aceita mais de um child, vamos usar Column: 

body: const Column(
  children: [
	  Text('Estudar para prova de matemática.'),
	  Text('Campeonato de futebol.'),
	  Text('Festa da Joana.'),
	],
),

Vamos adicionar ícones com Icon: 

body: const Column(
	  children: [
	    Icon(Icons.check_box_outline_blank),
	    Text('Estudar para prova de matemática.'),
...

1.5. Row

Para deixar os ícones referentes às tarefas na mesma linha das tarefas vamos usar Row: 

Row(
	children: [
		Icon(Icons.check_box_outline_blank),
	  Text('Estudar para prova de matemática.'),
	],
),

Para adicionar espaço entre o ícone e o texto podemos usar SizedBox(): 

Row(
	children: [
		Icon(Icons.check_box_outline_blank),
		SizedBox(width: 20,),
	  Text('Estudar para prova de matemática.'),
	],
),

1.6. Exercício

  1. Adicione um ícone para deixar Campeonato de futebol marcado como feito.
  2. Adicione um ícone de relógio para a atividade Festa da Joana.

1.7. Align

Se adicionarmos uma data em cada atividade, as datas ficam grudadas no texto da atividade. Podemos usar Align para alinhar a data no lado direito: 

Align(
	alignment: Alignment.centerRight,
	child: Text('16/08/2024'),
),

O alinhamento não altera o tamanho do elemento. Para aumentar o tamanho do elemento e destacar a data podemos usar Container ajustando width: 

Container(
	width: 200,
	child: Align(
		alignment: Alignment.centerRight,
		child: Text('21/08/2024'),
	),
),

Outra opção é ocupar todo o espaço disponível usando Expanded. 

Expanded(
  child: Align(
		alignment: Alignment.centerRight,
		child: Text('21/08/2024'),
	),
),

1.8. Exercício

  1. Adicione datas nas atividades: Estudar 16/08/2024, Campeonato 14/08/2024, Festa 23/08/2024.

1.9. Tamanho do texto

A propriedade style de Text controla o estilo recebendo um TextStyle. As propriedades incluem cor do texto, tamanho, família da fonte. 

appBar: AppBar(
	title: const Text(
	  'Atividades',
	  style: TextStyle(
      fontFamily: 'Quicksand',
      fontSize: 34,),
	),
),

1.10. Exercício

  1. Adicione style em todos os textos. Use a cor do texto azul para atividades que ainda vão acontecer, texto preto para o que já aconteceu, e a cor verde para atividades que vão acontecer mas já estão marcadas como feitas.
  2. Ajuste a fonte de todos os textos para 'Quicksand'. Ajuste o tamanho da fonte da atividade para 26 e o tamanho da fonte da data para 22.
  3. Pesquise como alterar a cor do ícone, ajuste a cor dos ícones para a mesma cor do texto da atividade.
  4. Pesquise como marcar um texto com linethrough, adicione uma atividade que foi cancelada e use o estilo linethrough.
