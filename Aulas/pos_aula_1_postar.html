2. Hello World

  • Função main()
  • runApp()

Toda aplicação Dart inicia em uma função main().

O framework Flutter fornece o método runApp() que recebe um Widget e o mostra na tela.

Pergunta Gemini: 

O que o método runApp() faz no Flutter?

O método runApp() é o ponto inicial da execução de uma aplicativo Flutter. Você informa um Widget que será a raiz da app.

Em Flutter podemos criar Widgets com estado (Statefull) ou sem estado (Stateless).

Pergunta Gemini: 

O que é um StatelessWidget em Flutter?

Vamos deixar o arquivo main.dart assim: 

import 'package:flutter/material.dart';

void main() {
  // início de Dart
  Widget app = const MinhaApp();
  // início do Flutter:
  runApp(app);
}

class MinhaApp extends StatelessWidget {
  const MinhaApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const Placeholder();
  }
}

2.2. Material Design

Material Design é um sistema para criar produtos com design bonito.

Em Flutter, MaterialApp é o Widget que implementa Material Design nas aplicações.

No começo MaterialApp ajusta um estilo feio para avisar que o estilo ainda não foi definido. Quem define o estilo na app é Scaffold. 

import 'package:flutter/material.dart';

void main() {
  // início de Dart
  Widget app = const MinhaApp();
  // início do Flutter:
  runApp(app);
}

class MinhaApp extends StatelessWidget {
  const MinhaApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: Text(
	'Hello World!',
	textDirection: TextDirection.ltr,
      ),
    );
  }
}

MaterialApp configura o Navigator que controla qual página está sendo exibida. O atributo home de MaterialApp ajusta a rota inicial da app.

O widget Scaffold implementa a estrutura de layout do Material Design. Agora colocando Scaffold o visual já melhora, a propriedade body de Scaffold define o conteúdo principal.

2.3. Exercício

  1. Usando o Gemini, crie uma app cartão de visitas, use uma pergunta do tipo: "Como criar uma app Flutter simples para mostrar um cartão de visitas contendo apenas os seguintes textos: Nome, Cargo, Empresa, E-mail". A app deve ficar assim:

Use seu nome ao invés de Fulano de Tal, seu prontuário ao invés de BA919191 e seu e-mail ao invés de fulano@ifsp.edu.br

2.3.1. Solução

import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Meu Cartão de Visitas',
      home: Scaffold(
	appBar: AppBar(
	  title: const Text('Meu Cartão'),
	),
	body: Center(
	  child: Column(
	    mainAxisAlignment: MainAxisAlignment.center,
	    children: const [
	      Text('Seu Nome', style: TextStyle(fontSize: 24)),
	      Text('Seu Cargo'),
	      Text('Sua Empresa'),
	      Text('seu.email@exemplo.com'),
	    ],
	  ),
	),
      ),
    );
  }
}

2.4. Stateful

Até agora nada muda em nossas apps, mas se precisamos alterar uma variável e mudar a tela por conta do novo valor devemos usar StatefulWidget. 

import 'package:flutter/material.dart';

class ComEstado extends StatefulWidget {
  const ComEstado({super.key});

  @override
  State<ComEstado> createState() => _ComEstadoState();
}

class _ComEstadoState extends State<ComEstado> {
  var nome = 'Mauro';
  var x = 1;

  void muda() {
    print(nome);
    // mudança:
    setState(() {
      x = x + 1;
      nome = 'Fulano $x';   
    });
    print(nome);
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
	body: Center(
	  child: Column(
	    children: [
	      Text('Hello $nome', style: const TextStyle(fontSize: 32),),
	      ElevatedButton(onPressed: muda, child: const Text('Altera nome'))
	    ],
	  ),
	),
      ),
    );
  }
}
